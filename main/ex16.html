모달창

모달창 사용 방법

data 속성으로(Via data attributes)


자바스크립트 없이 모달을 활성화합니다. 버튼같은, 콘트롤러 태그에 data-toggle="modal" 을 추가하고, data-target="#foo" 나 href="#foo" 와 같이 토글할 특정한 모달을 정해주면 됩니다.
버튼의 data-target="#myModal"과 모달창의 id="myModal"가 일치해야한다.
모달창의 크기 옵션 div.modal-dialog 위치에 modal-lg, modal-sm 옵션을 주면 크고 작은 모달창으로 적용된다.
모달창 생성시 애니메이션 제거는 div.modal 위치의 .fade를 제거하면 된다.

자바스크립트로


$('#myModal').modal(options);
더 자세한 사항은 부트스트랩 사이트에서 확인 가능합니다.
